# AI Policy

## Accountability

You are responsible for every line of code you submit, regardless of how it was produced. If you use AI tools, you must review, understand, and verify all output before committing it.

## AI Tool Usage

AI tools are permitted for code generation, review, and documentation.

## AI-Generated Code

AI-generated code is accepted. No special attribution or review process is required beyond the standard contribution workflow.

## AI in CI/CD

AI tools may be used in CI/CD pipelines for code analysis, test generation, and deployment.

## Training Data

This project does not opt out of AI training data collection. Standard platform terms apply.

## Licensing

AI-generated contributions must be compatible with this project's license. You must be able to certify your submissions under the project's contribution terms.

## Policy Evolution

This policy will change as the tooling does. PRs welcome.
